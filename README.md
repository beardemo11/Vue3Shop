# 隨心飛旅遊景點購物網站 Vue CLI 3

## Demo

Vue CLI 3
https://beardemo11.github.io/Vue3Shop/#/

## 簡介

使用 Vue Cli 3 、 Vuex 及 Vue Router 製作的購物網站，主要功能有

前台:
* 首頁 Banner 、 商品 輪播
* 購物車功能
* 熱門景點
* 我的最愛功能 (使用 localStorage)
* 商品列表、分類展示
* 單一商品詳細介紹
* 訂單結帳
* 優惠卷套用
* 結帳表單驗證
* 訂閱我們
* 寄送電子郵件

後台:
* 商品管理
* 客戶訂單管理
* 優惠卷管理

## 技術
* Vue Cli 3
* Vuex
* Vue Router
* Vue Component
* API操作(前後端分離)
* Webpack 打包
* ESlint
* Bootstrap5
* localStorage
